ANTHROPIC PRESENTA CLAUDE OPUS 4.5 CON VENTANA DE 1M TOKENS

01/04/2026 // ▓ 1 MIN LECTURA // FUENTE: MANUAL

Anthropic lanzó Claude Opus 4.5, la nueva versión de su modelo más avanzado. Las mejoras clave incluyen una ventana de contexto de 1 millón de tokens, mejor rendimiento en tareas de programación con 95% en HumanEval y razonamiento matemático con 89% en MATH-500. El modelo está disponible en claude.ai para usuarios Pro y Team, y vía API con precios competitivos de 15 dólares por millón de tokens de input y 75 dólares por millón de output. Las mejoras en capacidades agent lo posicionan como competidor directo de GPT-6 en flujos de trabajo automatizados.
