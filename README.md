# ABISMOX - Gaming + Tech News Archive

Blog automático de noticias gaming + tecnología con diseño retro N64/PS1.

## Características

- Hero section con juego Snake en background borroso
- Título ABISMOX en estilo pixel art
- Feed dinámico de noticias cargado desde JSON
- Filtros por categoría (Videojuegos, Tecnología)
- Diseño retro con colores vibrantes
- Responsive (mobile-first)
- 100% vanilla JavaScript (sin frameworks)

## Estructura de archivos

```
Blogx/
├── index.html          # Estructura principal del blog
├── style.css           # Estilos retro (N64/PS1)
├── main.js             # Carga y renderizado de noticias
├── snake.js            # Juego Snake decorativo para el hero
└── data/
    └── noticias.json   # Base de datos de noticias
```
